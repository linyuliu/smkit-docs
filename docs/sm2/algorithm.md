# SM2 算法原理

深入了解 SM2 椭圆曲线公钥密码算法的数学原理和实现细节。

## 椭圆曲线基础

椭圆曲线方程：

$$
y^2 = x^3 + ax + b \pmod{p}
$$

::: info 椭圆曲线性质
- 点加法运算构成Abel群
- 基于离散对数难题
- 提供高效的公钥密码系统
:::

## 密钥生成

随机选择私钥 $d \in [1, n-1]$，计算公钥：

$$
P = [d]G
$$

其中 $G$ 是基点。

## 数字签名

签名算法 SM2-DSA 基于 ECDSA 改进而来。

## 公钥加密

加密算法基于椭圆曲线点乘和密钥派生。
