# Java 安装配置

如何安装和配置 SMKit Java 库。

## Maven 依赖

```xml
<dependency>
    <groupId>cn.smkit</groupId>
    <artifactId>smkit-core</artifactId>
    <version>1.0.0</version>
</dependency>
```

## Gradle 依赖

```groovy
implementation 'cn.smkit:smkit-core:1.0.0'
```

## 系统要求

- JDK 8 或更高版本
- Maven 3.6+ 或 Gradle 6.0+
